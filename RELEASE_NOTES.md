# 햄토리 v0.1.3

배포일: 2026-05-27

고정 지출/수입을 더 현실적으로 관리할 수 있도록 다듬은 업데이트입니다. 할부처럼 기간이 정해진 내역은 종료월까지만 반영하고, 월급처럼 계속 이어지는 내역은 종료월 없이 반복되도록 설정할 수 있습니다.

## 좋아진 점

- 고정내역에 `반영 시작월`과 `반영 종료월`을 설정할 수 있습니다.
- `종료월 없이 계속 반복` 옵션을 추가했습니다.
- 6개월 할부처럼 기간이 정해진 지출은 종료월까지만 달력에 자동 반영됩니다.
- 월급처럼 기한이 없는 수입/지출은 계속 반복되도록 등록할 수 있습니다.
- 고정내역을 수정할 때 선택한 반영 시작월부터 새 금액과 내용으로 적용됩니다.
- 기존 달에 이미 생성된 고정내역은 수정 시점 기준으로 다시 정리되어 중복이나 오래된 금액이 남지 않도록 했습니다.

## 달력 사용성 개선

- 날짜별 수입/지출 내역에서 바로 삭제할 수 있습니다.
- 달력 오른쪽 입력/내역 영역을 고정하고, 내역이 많아지면 내부에서 스크롤되도록 정리했습니다.
- 선택한 날짜의 수입/지출 내역을 더 안정적인 카드 형태로 보여줍니다.

## 고정내역 화면 개선

- 입력 영역과 등록된 고정내역 영역의 높이를 맞춰 화면 균형을 정리했습니다.
- 등록된 고정내역이 많아지면 목록 안에서만 스크롤됩니다.
- 고정내역 카드에 적용 시작월과 종료월이 표시됩니다.

## 업데이트 안정성

- 기존 데이터베이스도 새 고정내역 기간 기능을 자동으로 사용할 수 있게 정리했습니다.
- 포터블 업데이트에 필요한 파일 구성을 보강했습니다.
- 기존 가계부 데이터와 Google Drive 연결 정보는 사용자 데이터 영역에 유지되도록 업데이트 구조를 맞췄습니다.

# 햄토리 v0.1.2

배포일: 2026-05-26

매월 반복되는 수입과 지출을 따로 관리할 수 있게 만든 업데이트입니다.

## 새로 추가된 기능

- `고정내역` 메뉴가 추가되었습니다.
- 월급, 월세, 통신비, 구독료처럼 매월 같은 날 발생하는 수입/지출을 등록할 수 있습니다.
- 등록한 고정내역은 달력에서 해당 날짜에 자동으로 표시됩니다.
- 31일로 등록한 항목은 2월처럼 날짜가 짧은 달에는 그 달의 마지막 날에 표시됩니다.

## 사용하기 쉬워진 점

- 고정내역은 수입과 지출 색상으로 구분되어 한눈에 볼 수 있습니다.
- 고정내역 목록은 항목이 많아져도 목록 안에서 스크롤됩니다.
- 고정내역을 수정하면 달력에 자동 반영된 기존 내역이 새 설정 기준으로 다시 정리됩니다.
- 자동 반영된 내역을 달력에서 삭제하면 그 달의 해당 내역은 다시 생기지 않습니다.

## 업데이트와 포터블 패키지

- 웹 설정 화면에서 업데이트를 진행하는 방식에 맞춰 배포 파일을 정리했습니다.
- 업데이트 파일 구조는 `_update_payload` 기준으로 고정했습니다.
- 혹시 폴더 구조가 꼬였을 때 사용할 수 있는 복구 파일도 함께 포함됩니다.

# 햄토리 v0.1.1

배포일: 2026-05-26

종료 파일이 안정적으로 동작하도록 수정한 패치 버전입니다.

## 좋아진 점

- 햄토리 종료 파일을 실행했을 때 앱이 제대로 닫히도록 수정했습니다.
- 종료 처리 과정을 별도 스크립트로 분리해 실행 환경에 덜 흔들리게 만들었습니다.
- 업데이트용 포터블 파일을 공개 릴리즈 저장소에서 받을 수 있도록 정리했습니다.

# 햄토리 v0.1.0

배포일: 2026-05-26

햄토리를 처음 사용할 수 있는 버전입니다. 달력에 수입과 지출을 기록하고, 대시보드에서 한 달 흐름을 확인하고, 필요하면 Google Drive로 집과 회사 데이터를 맞출 수 있게 준비했습니다.

## 이번 버전에서 좋아진 점

- 앱 이름과 화면 구성을 햄토리에 맞게 정리했습니다.
- 대시보드, 달력, 설정을 분리해 원하는 기능을 찾기 쉽게 만들었습니다.
- 전체 화면은 밝고 깔끔한 느낌으로 정리하고, 입력 영역은 읽기 편하게 다듬었습니다.
- 수입과 지출 금액은 서로 다른 색상으로 표시해 한눈에 구분할 수 있습니다.

## 달력에서 바로 기록

- 달력에서 날짜를 선택한 뒤 바로 수입과 지출을 입력할 수 있습니다.
- 선택한 날짜의 수입 합계와 지출 합계를 따로 볼 수 있습니다.
- 금액 입력 시 3자리마다 쉼표가 자동으로 표시됩니다.
- 저장이 완료되면 입력칸이 비워져 다음 내역을 바로 작성할 수 있습니다.

## 대시보드 개선

- 연도별, 월별 흐름을 확인할 수 있습니다.
- 지출 카테고리별 금액과 비율을 함께 볼 수 있습니다.
- 거래내역과 카테고리 영역은 내용이 많아져도 화면 안에서 스크롤해 볼 수 있습니다.
- 카테고리별 지출 총액과 거래 건수를 더 명확하게 표시했습니다.

## 카테고리와 스티커

- 설정에서 카테고리를 직접 추가하고 수정할 수 있습니다.
- 카테고리마다 어울리는 아이콘을 선택할 수 있습니다.
- 달력에 스티커를 붙여 특별한 날을 꾸밀 수 있습니다.
- 스티커는 달력 안에서만 사용할 수 있고, 월별로 따로 저장됩니다.
- 스티커 크기 조절과 회전을 지원합니다.

## Google Drive 동기화

- Google Drive를 연결하면 여러 PC에서 같은 햄토리 데이터를 사용할 수 있습니다.
- 자동 동기화를 켜두면 앱을 사용할 때 데이터를 맞춰줍니다.
- 마지막 동기화 시간을 보기 쉬운 형식으로 표시합니다.
- 필요할 때 지금 동기화 버튼으로 바로 데이터를 맞출 수 있습니다.

## 업데이트와 릴리즈 노트

- 설정에서 현재 버전과 업데이트 가능 여부를 확인할 수 있습니다.
- 새 버전이 있으면 설정 화면에서 업데이트를 적용할 수 있습니다.
- 릴리즈 노트 버튼으로 버전별 변경 내역을 언제든지 다시 볼 수 있습니다.
- 업데이트해도 기존 가계부 데이터는 유지됩니다.

## 사용 안내

- 압축을 풀고 햄토리 실행 파일을 열면 바로 사용할 수 있습니다.
- 햄토리를 종료해야 할 때는 함께 제공되는 종료 파일을 사용할 수 있습니다.
